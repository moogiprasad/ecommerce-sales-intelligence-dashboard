# 📊 Executive Sales Intelligence Dashboard

An end-to-end Business Intelligence project that transforms raw e-commerce transaction data into actionable business insights using **Python**, **Pandas**, **Power BI**, and **DAX**.

## 📌 Project Overview

This project analyzes an online retail e-commerce dataset and presents key business metrics through an interactive Power BI dashboard. The workflow includes data cleaning, feature engineering, exploratory data analysis (EDA), KPI creation, and dashboard development to support data-driven decision-making.

The dashboard helps stakeholders monitor:

- 💰 Revenue Performance
- 👥 Customer Behavior
- 🌍 Country-wise Sales Distribution
- 📈 Monthly Revenue Trends
- 📦 Product & Sales Performance

---

## 🎯 Objectives

- Clean and preprocess raw e-commerce transaction data
- Generate meaningful business KPIs
- Analyze customer purchasing patterns
- Identify top-performing countries and customers
- Visualize monthly sales trends
- Build an interactive executive dashboard using Power BI

## 🧹 Data Preprocessing

The following preprocessing steps were performed:

- Removed duplicate records
- Removed missing Customer IDs
- Filtered cancelled and returned transactions
- Removed invalid Unit Prices
- Converted Invoice Date to DateTime format
- Created a new **Revenue** feature
- Extracted:
  - Year
  - Month
  - Day
  - Hour

## 💡 Key Business Insights

- The **United Kingdom** contributes the highest share of total revenue.
- Monthly revenue exhibits seasonal variations, indicating periods of stronger customer demand.
- A relatively small group of customers contributes a significant portion of overall sales.
- Revenue is concentrated in a few key markets, highlighting opportunities for targeted business expansion.
